



Here's an organized structure project files. I've created a clear directory structure and improved naming conventions for better readability and maintenance.

Project Directory Structure

/project-root
│
├── /static
│   ├── /css
│   │   ├── assessment.css
│   │   ├── login.css
│   │   ├── register.css
│   │   ├── styles.css
│   │
│   ├── /java_script
│   │   ├── assessment.js
│   │   ├── image_switcher.js
│   │
│   ├── /images
│   │   ├── image2.png
│   │   ├── image.png
│   │   ├── hero-image.png
│   │
│   ├── /videos
│   │   └── mental-health.mp4
│
└── /templates
    ├── index.html
    ├── login.html
    ├── register.html
    ├── assessment.html


Explanation

/static Directory:

/css Subdirectory: Contains all CSS files.

assessment.css: Styles for the assessment pages.

login.css: Styles for the login page.

register.css: Styles for the register page.

styles.css: General styles for the landing (index) page.


/js Subdirectory: Contains all JavaScript files.

assessment.js: JavaScript for the assessment pages.

image_switcher.js: JavaScript for changing images on the login and register pages.


/images Subdirectory: Contains all image files.

image2.png: Second image for the login page.

image.png: First image for the login page.

hero-image.png: Image for the landing (index) page.


/videos Subdirectory: Contains all video files.

mental-health.mp4: Video for the landing (index) page.

/templates Directory: Contains all HTML files.

index.html: Landing page.

login.html: Login page.

register.html: Register page.

assessment.html: Assessment page.
